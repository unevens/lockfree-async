Detailed Description

template<typename T = char>
class lockfree::Messenger< T >

Wrapper around QueueWorld's QwMpmcPopAllLifoStack with functionality to allocate and recycle nodes.

Template Parameters
Tthe type of the data held by the nodes, char is used as default for Messages that are just notification and do not need to have data.

Receives the message that was sent for last.

Non blocking. If there is any message not yet received, "ReceiveLast" will move the one that was sent for last to its argument, and return true; otherwise it will return false. REMARK: When "receiveLastMessage" is called, ONLY the message sent for last is received, all the others are discarded.

Parameters
messagean already allocated message to which the received message will be moved.
