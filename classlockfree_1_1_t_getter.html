Detailed Description

template<class TClassToGet, class TClassToStore>
class lockfree::TGetter< TClassToGet, TClassToStore >

A class to get a thread-local copy of the object stored in the Async class, or of an object of an other type constructed from it.

Non-blocking version.

Template Parameters
TClassToStorethe type of the object stored in the Async object.
TClassToGetthe type of object to be returned by the method get of the getter. It must be constructible from a constant reference to a TClassToStore object.
